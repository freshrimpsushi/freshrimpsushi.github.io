벡터 공간의 리플렉시브

770 함수해석  작성일: 2018-12-13 
작성자: 류대식 사장  

목차
  • 증명[1]

$X^{} \approx X$ 면 $X$ 가 리플렉시브 하다고 한다.일반적으로 스페이스는 듀얼을 취할때마다 점점 그 크기가 더 커진다.그런데 리플렉시브라는 말은 사실상 듀얼 스페이스가 계속해서 커지지 않는 공간이라고 보아도 좋다.리플렉시브한 공간에는 다음의 예시가 있다.[1]**  $1 < p < \infty$ 에 대해 ${\mathcal{l}^{p}}^{**} \approx \mathcal{l}^{p}$**[2]**  $\dim X = n$ 이면 $X^{**} \approx X$**[3]**  $H$ 가 힐베르트 공간이면 $H^{**} \approx H$

증명[1]

우선은 $\displaystyle {{1} \over {p}} + {{1} \over {q}} = 1$ 에 대해 ${\mathcal{l}^{p}}^{ * } \approx \mathcal{l}^{q}$ 임을 보일 것이다.사상 $\phi : {\mathcal{l}^{p}}^{ * } \to \mathcal{l}^{q} $ 를 $f \in {\mathcal{l}^{p}}^{ * } $ 에 대해 $\phi (f) = \left( f(e_{1}) , f(e_{2}) , \cdots \right)$ 이라고 정의하자. 여기서 $e_{j}$ 는 $j$ 번째 성분만 $1$ 이고 나머지는 $0$ 인 단위벡터 $e_{j}:=(\cdots ,0, 1 , 0 , \cdots )$ 를 의미한다.이제 복소수의 부호 $\text{sign}$ 이 포함된 $\displaystyle y_{j} : = {{ \text{sign} \left( f^q (e_{j} ) \right) f^{q-1} (e_{j} ) } \over { \left( \sum_{j=1}^{n} \left| f (e_{j} ) \right|^{q} \right)^{{{1} \over {p}}} }} \in \mathbb{C}$ 을 정의하자.모든 $\lambda \in \mathbb{C}$ 에 대해 $|\lambda| = 1$ 이므로 $$ \displaystyle | y_{1} e_{1} + \cdots + y_{n} e_{n} |_{p}^{p} = \sum_{j=1}^{n} |y_{j}|^{p} = \sum_{j=1}^{n} {{ 1 \cdot \left| f (e_{j} ) \right|^{(q-1)p} } \over { \left( \sum_{j=1}^{n} \left| f (e_{j} ) \right|^{q} \right) }} $$ $(q-1)p = q$ 이므로 $$ \displaystyle | y_{1} e_{1} + \cdots + y_{n} e_{n} |_{p}^{p} = \left| f (e_{j} ) \right|^{q} \right)^{{{1} \over {p}}} }} \left( \text{sign} \left( f^q (e_{1} ) \right) f^{q-1} (e_{1} ) f(e_{1})  + \cdots +\text{sign} \left( f^q (e_{n} ) \right) f^{q-1} (e_{n} ) f(e_{n}) \right) $$ \displaystyle  =   {{ 1 } \over { \left( \sum_{j=1}^{n} \left| f (e_{j} ) \right|^{q} \right)^{{{1} \over {p}}} }} \left( \text{sign} \left( f^q (e_{1} ) \right) f^{q} (e_{1} )+ \displaystyle = {{ 1 } \over { \left( \sum_{j=1}^{n} \left| f (e_{j} ) \right|^{q} \right)^{{{1} \over {p}}} }} \left( \left| f (e_{1})  \right|^{q} + \cdots + \left| f (e_{n})  \right|^{q} \right) $$ \displaystyle = \left( \sum_{j=1}^{n} \left| f(e_{j} ) \right|^{q} \right)^{1- {{1} \over {p}} } $$ \displaystyle = \left( \sum_{j=1}^{n} \left| f(e_{j} ) \right|^{q} \right)^{ {{1} \over {q}} } $**Part 1.** $\displaystyle \left| \left( \sum_{j=1}^{\infty} \left| f(e_{j} ) \right|^{q} \right)^{ {{1} \over {q}} } \right| \le | f |   $$ f$ 는 유계이므로 모든 $n \in \mathbb{N}$ 에 대해 $$ \displaystyle \left| \left( \sum_{j=1}^{n} \left| f(e_{j} ) \right|^{q} \right)^{ {{1} \over {q}} } \right| \le | f | | y_{1} e_{1} + \cdots + y_{n} e_{n} |_{p}  $$ $\displaystyle | y_{1} e_{1} + \cdots + y_{n} e_{n} |_{p}^{p} = 1$ 이므로 정리하면 $$ \displaystyle \left| \left( \sum_{j=1}^{\infty} \left| f(e_{j} ) \right|^{q} \right)^{ {{1} \over {q}} } \right| \le | f |   $$ **Part 2.**  $\phi$ 는 함수다.위의 **Part 1.** 에서 $\displaystyle \left| \phi (f) \right| = \left| \left( \sum_{j=1}^{\infty} \left| f(e_{j} ) \right|^{q} \right)^{ {{1} \over {q}} } \right| \le | f | < \infty $ 이므로 $ \phi ( f \phi (f_{0} ) = \left( f_{0} (e_{1}) , f_{0} (e_{2}) , \cdots \right) = \left( \sum_{j=1}^{\infty} e_{1} \lambda_{j} , \sum_{j=1}^{\infty} e_{2} \lambda_{j} , \cdots \right) = (\lambda_{1} ,
