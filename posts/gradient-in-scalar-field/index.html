스칼라 필드의 그래디언트

1010 벡터 해석  작성일: 2019-04-28 
작성자: 류대식 사장  

벡터장의 미분계수 : 야코비 행렬스칼라 필드 $f : \mathbb{R}^{n} \to  \mathbb{R}$ 에 대해 한 점 $\mathbb{x}{0} \in \mathbb{R}^{n}$ 에서 함수의 증가율이 가장 큰 방향을 나타낸 벡터를 $\mathbb{x}{0}$ 에서 $f$ 의 그래디언트 라고 한다.스칼라장의 예시로써 위의 그림을 생각해보자. 위 그림은 $z(x,y) = x^2 - y^2$ 와 같이 정의된 함수 $z : \mathbb{R}^{2} \to \mathbb{R}$ 을 시각적으로 나타낸 것이다. $y= f(x)$ 와 같은 곡선에서는 변화율을 구할만할 방향이 $x$ 축 뿐이었기 때문에 접선의 기울기를 $f$ 의 그래디언트로 정의할 수 있었지만, 위와 같은 곡면은 무한히 많은 방향에 대해 변화율이 제각각 다를 수 있기 때문에 별도로 정의한 것이다.한편 $f : \mathbb{R} \to \mathbb{R}$ 일 때 $x=a$ 에서 미분계수가 존재한다면 $f'(a)$ 는 $x=a$ 에서 $f$ 의 그래디언트가 된다. $$  \displaystyle \nabla f : = \left({{ \partial f} \over { \partial x_{1} }} , \cdots , {{ \partial f} \over { \partial x_{n} }} \right) $$ 와 같이 정의된 $\nabla$ 를 **델 연산자** 라고하는데, $\nabla f ( x_{0} ) $ 는 $x_{0} \in \mathbb{R}^{n}$ 에서 스칼라장 $f : \mathbb{R}^{n} \to \mathbb{R}$ 의 미분계수라고 할 수 있으며 재미있게도 다음과 같은 정리가 성립한다. [ **NOTE** : $\nabla$ 기호 자체는 나블라Nabla라고 읽는다. ]$\mathbb{x}_{0} \in \mathbb{R}^{n}$ 의 근방에서 두 번 미분가능한 $f : \mathbb{R}^{n} \to \mathbb{R}$ 의 그래디언트는 $\nabla f \left( \mathbb{x}_{0} \right)$

증명

$| \mathbb{d} | = 1$ 이 되도록하는 방향 벡터 $\mathbb{d} : = ( d_1 , \cdots , d_n ) $ 을 정의하자.다변수 함수의 테일러 정리에 의해 $$ \displaystyle f \left( x_{0} + h \mathbb{d} \right) = f ( \mathbb{x}_{0} ) + h \left[  {{ \partial f ( \mathbb{x}_{0} ) } \over { \partial x_{1} }} d_{1} + \cdots + {{ \partial f ( \mathbb{x}_{0} ) } \over { \partial x_{n} }} d_{n} \right] + O (h^2) $$ 행렬 꼴로 바꾸면 $$ \displaystyle f \left( x_{0} + h \mathbb{d} \right) - f ( \mathbb{x}_{0} ) = h \begin{bmatrix} {{ \partial f ( \mathbb{x}_{0} ) } \over { \partial x_{1} }} \\ \vdots \\ {{ \partial f ( \mathbb{x}_{0} ) } \over { \partial x_{n} }} \end{bmatrix} \cdot \begin{bmatrix} d_{1} \\ \vdots \\ d_{n}  \end{bmatrix} + O (h^2) $$ 벡터 꼴로 바꾸면 $$ \displaystyle {{ f \left( x_{0} + h \mathbb{d} \right) - f ( \mathbb{x}_{0} )} \over {h}} = \nabla f \left( \mathbb{x}_{0} \right) \cdot \mathbb{d} + O (h) $$ $h \to \infty$ 일 때 $$ \displaystyle \nabla f \left( \mathbb{x}_{0} \right) \cdot \mathbb{d} = \lim_{h \to \infty} {{ f \left( x_{0} + h \mathbb{d} \right) - f ( \mathbb{x}_{0} )} \over {h}} $$ $\mathbb{b}$ 가 $\mathbb{x}_{0}$ 에서 $f$ 의 기울기와 같은 방향이라는 것은 $\mathbb{d}$ 가 $$ \displaystyle \lim_{h \to \infty} {{ f \left( x_{0} + h \mathbb{d} \right) - f ( \mathbb{x}_{0} )} \over {h}} $$ 이 가장 커지도록 하는 단위벡터라는 뜻이다. 이를 만족하는 경우는 $\displaystyle \mathbb{d} = {{\nabla f \left( \mathbb{x}_{0} \right) } \over { \left| \nabla f \left( \mathbb{x}_{0} \right) \right| }} $ 뿐이고, 따라서 $$ \displaystyle \nabla f \left( \mathbb{x}_{0} \right) = \left| \nabla f \left( \mathbb{x}_{0} \right) \right| \mathbb{d} $$ 은 $\mathbb{x}_{0} $ 에서 $f$ 의 그래디언트가 된다.
