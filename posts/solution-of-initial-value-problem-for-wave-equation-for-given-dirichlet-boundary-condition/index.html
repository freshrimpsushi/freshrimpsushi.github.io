디리클레 경계 조건이 주어진 파동방정식에 대한 초기값 문제의 풀이

583 편미분방정식  작성일: 2018-06-20 
작성자: 류대식 사장  


🚧 이 포스트는 아직 이관 작업이 완료되지 않았습니다 🚧

$\begin{cases} u_{tt} = c^2 u_{xx} \\ u(0,x) = f(x) \\ u_{t}(0,x) = g(x) \\ \end{cases}$

주어진 방정식은 파동방정식이 길이가 $l$ 인 $1$차원 공간 상의 디리클레 경계 조건 $\begin{cases} u(t,0) = \alpha(t)  \\ u(t,l) = \beta (t) \end{cases}$ 이 $\alpha = \beta = 0$ 으로 주어지고 파형에 대한 초기 조건이 있는 경우다. 이러한 문제 유형 중에는 가장 쉽고 단순한 형태다.여기서 $t$ 는 시간, $x$ 는 위치, $u(t,x)$ 는 시간 $t$ 일 때의 파형을 나타낸다.$f$ 와 $g$ 는 초기 조건으로써 특히 $f$ 는 $t=0$ 일 때의 파형을 나타낸다.경계 조건이 주어진 경우 달랑베르의 공식은 사용할 수 없게 되며, 열방정식을 풀 때와 비슷한 아이디어를 사용하게 된다.**풀이 **Step 1. 해가 $u(t,x) = w(t) v(x) $ 로 나타난다고 가정해보면 파동방정식을 풀어야하므로 $w''(t) v(x) = c^2 w(t) v''(x)$보기 좋게 정리하면 $\displaystyle {{w''(t)} \over {w(t) } } v(x) = c^2  {{v''(x)} \over  {v(x)}} = \lambda$여기서 $\displaystyle {{\partial } \over { \partial x }} \lambda = {{\partial } \over { \partial x }} \left( {{w''(t)} \over {w(t) } } \right) =  0 $ 이고 $\displaystyle {{\partial } \over { \partial t }} \lambda = {{\partial } \over { \partial x }} \left( c^2 {{v''(x)} \over {v(x) } } \right) =  0 $ 이므로 $\lambda$ 는 상수다.**Step 2. $\lambda$ 가 상수임이 보장되었으므로, 이계미분방정식 $w'' - \lambda w = 0$ 와 $\displaystyle v'' - {{\lambda} \over {c^2}}  v = 0$ 를 각자 풀면 된다.해들은 열방정식을 풀 때와는 달리 $\lambda$ 앞의 부호가 다르므로 $\lambda <0$ 일 때 비자명해가 될 것이다. $\displaystyle \omega := {{ n \pi c} \over {l}} $ 에 대해 해는 위의 형태로 나타난다.특히 방정식을 푸는 기본해는 $\displaystyle u_{n}(t,x) = \cos {{n \pi c t} \over {l}} \sin {{ n \pi x} \over {l}}$ 과 $\displaystyle \tilde{u} _{n}(t,x) = \sin {{n \pi c t} \over {l}} \sin {{ n \pi x} \over {l}} $ 이다.따라서 해는 어떤 $b_{n}, d_{n}$ 에 대해 $\displaystyle u(t,x) = \sum_{n = 1}^{\infty} \left[ b_{n } \cos {{n \pi c t} \over {l}} \sin {{ n \pi x} \over {l}} + d_{n} \sin {{n \pi c t} \over {l}} \sin {{ n \pi x} \over {l}} \right]$ 로 나타난다.**Step 3. 초기 조건에 대한 푸리에 전개 $\displaystyle u(0,x) = \sum_{n=1}^{\infty} \left[ b_{n } \sin {{ n \pi x} \over {l}} \right] = f(x) $ 이므로 $\displaystyle b_{n} = \left< f(x) , \sin {{n \pi x } \over {l}} \right> = {{2} \over {l}} \int_{0}^{l} f(x) \sin {{n \pi x} \over {l}} dx $$ \displaystyle u_{t}(0,x) = \sum_{n=1}^{\infty} \left[ d_{n } {{n \pi c} \over {l}} \sin {{ n \pi x} \over {l}} \right] = g(x) $ 이므로 $\displaystyle d_{n} = {{l} \over {n \pi c}} \left< g(x) , \sin {{n \pi x } \over {l}} \right> = {{2} \over { n \pi c }} \int_{0}^{l} g(x) \sin {{n \pi x} \over {l}} dx$따라서 $\displaystyle u(t,x) = \sum_{n=1}^{\infty} \left[ {{2} \over {l}} \int_{0}^{l} f(x) \sin {{n \pi x} \over {l}} dx \cos {{n \pi c t} \over {l}} \sin {{ n \pi x} \over {l}} + {{2} \over { n \pi c }} \int_{0}^{l} g(x) \sin {{n \pi x} \over {l}} dx \sin {{n \pi c t} \over {l}} \sin {{ n \pi x} \over {l}} \right] $
