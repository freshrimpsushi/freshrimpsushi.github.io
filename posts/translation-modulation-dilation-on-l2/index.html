L2 공간의 트랜슬레이션 모듈레이션 다일레이션

1616 함수해석  작성일: 2020-08-17 
작성자: 류대식 사장  

목차
  • 증명[1]**


🚧 이 포스트는 아직 이관 작업이 완료되지 않았습니다 🚧

1. $a \in \mathbb{R}$ 에 대해 다음과 같이 정의된 $T_{a} : L^{2} \to L^{2}$ 를 트랜슬레이션 이라 한다. $$ \left( T_{a} f \right) (x) := f(x-a) $$ 2. $b \in \mathbb{R}$ 에 대해 다음과 같이 정의된 $E_{b} : L^{2} \to L^{2}$ 을 모듈레이션 이라 한다. $$ \left( E_{b} f \right) (x) := e^{2 \pi i b x} f(x) $$ 3. $c > 0$ 에 대해 다음과 같이 정의된 $D_{c} : L^{2} \to L^{2}$ 을 다일레이션 이라 한다. $$ \left( D_{c} f \right) (x) := {{ 1 } \over { \sqrt{c} }} f \left( {{ x } \over { c }} \right) $$

위의 선형 작용소들은 $L^{2}$ 공간에서 흔히 쓰이는 선형 작용소들이다. 한국어로는 평행이동Translation, 변조Modulation, 팽창Dilation이라 번역할 수 있겠지만 영어를 그대로 읽는 것이 수식적으로 받아들이기 편할 것이다. 모듈레이션 에서 곱해진 $e^{2 \pi i b x}$ 는 단어 그대로 추상화된 회전이다. 다일레이션에서 곱해진 $\displaystyle {{ 1 } \over { \sqrt{c} }}$ 는 놈 $\left| \cdot \right|_{2} $ 에 맞추기 위해 루트가 씌워져있다고 보아도 무방하다. 특히 $c = 1/2$ 에 대해서는 다음과 같이 정의된 $D$ 가 특별한 역할을 하기도 한다. $$ ( D f ) (x) := \sqrt{2} f (2x)  $$ $D$ 는 편의를 위해서 $j \in \mathbb{Z}$ 에 대해 다음과 같이 쓰여진다. $$ ( D^{j} f ) (x) := \sqrt{2}^{j} f \left( 2^{j} x \right)  $$

모든 $a, b \in \mathbb{R}$, $c > 0$ 와 $f,g \in L^{1}$ 에 대해

[1]** $T_{a} , E_{b}, D_{c}$ 는 유계 선형 작용소다.**[2] 역작용소** : $T_{a} , E_{b}, D_{c}$ 는 유니터리다.**[3] 교환관계 :

$$ (T_{a} E_{b} f ) (x) = e^{- 2 \pi i b a} (E_{b} T_{a} f ) (x)  \\ (T_{a} D_{c} f ) (x) = (D_{c} T_{a/c} f ) (x) \\ (D_{c} E_{b} f ) (x) = (E_{b/c} D_{c} f ) (x)  $$ **[4] 푸리에 변환과의 관계 :

$$ \mathcal{F} T_{a} = E_{-a} \mathcal{F} \\ \mathcal{F} E_{b} = T_{b} \mathcal{F} \\ \mathcal{F} D_{c} = D_{1/c} \mathcal{F} $$

$D$ 에 대해서는 위 정리들의 따름정리로써 $j, k \in \mathbb{Z}$ 에 대해 다음을 얻을 수 있다. $$ T_{k} D^{j} = D^{j} T_{2^{j} k } \\ D^{j} T_{k} = T_{2^{-j}k} D^{j} \\ \left( D^{j} \right)^{ * } = D^{-j}  $$

증명[1]**

Part 1. 선형 모든 $f,g \in L^{2}$ 와 $\alpha , \beta \in \mathbb{C}$ 에 대해 $$ \begin{eqnarray*} T_{a} \left( \alpha f + \beta g \right)(x) &=& \left( \alpha f + \beta g \right)(x-a) \\ &=&  \alpha f (x-a) + \beta g (x-a) \\ &=& \alpha T_{a} f (x) + \beta T_{a} g (x) \end{eqnarray*} $$ 이므로 $T_{a}$ 는 리니어다. $$ \begin{eqnarray*} E_{b} \left( \alpha f + \beta g \right)(x) &=& e^{ 2 \pi i b x } \left( \alpha f + \beta g \right)(x) \\ &=&   \alpha e^{ 2 \pi i b x } f (x) + \beta  e^{ 2 \pi i b x } g (x) \\ &=& \alpha E_{b} f (x) + \beta E_{b} g (x) \end{eqnarray*} $$ 이므로 $E_{b}$ 는 리니어다. $$ \begin{eqnarray*} D_{c} \left( \alpha f + \beta g \right)(x) &=& {{ 1 } \over { \sqrt{c} }} \left( \alpha f + \beta g \right) \left( {{ x } \over { c }} \right) \\ &=&  \alpha {{ 1 } \over { \sqrt{c} }} f (x) + \beta {{ 1 } \over { \sqrt{c} }} g (x)  \\ &=& \alpha D_{c} f (x) + \beta D_{c} g (x) \end{eqnarray*} $$ 이므로 $D_{c}$ 는 리니어다. Part 2. 유계 $t := x - a$ 와 같이 치환하면 $$ \begin{eqnarray*} \left| T_{a} f \right|_{2}  &=& \int_{-\infty}^{\infty} \left| T_{a} f \left( x \right) \right|^{2} dx  \\ &=& \int_{-\infty}^{\infty} \left| f \left( x - a \right) \right|^{2} dx \\ &=& \int_{-\infty}^{\infty} \left| f \left( t \right) \right|^{2} dt \\ &=& \left| f \right|_{2}  \end{eqnarray*} $$ 이므로 $T_{a}$ 는 바운디드다. $\left| e^{2 \pi i b x } \right| =1 $ 이므로 $$ \begin{eqnarray*} \left| E_{b} f \right|_{2}  &=& \int_{-\infty}^{\infty} \left| E_{b} f \left( x \right) \right|^{2} dx  \\ &=& \int_{-\infty}^{\infty} \left| e^{2 \pi i b x } f \left( x \right) \right|^{2} dx \\ &=& \int_{-\infty}^{\infty} 1 \cdot \left| f \left( t \right) \right|^{2} dt \\ &=& \left| f \right|_{2} \end{eqnarray*} $$ 이므로 $E_{b}$ 는 바운디드다. $t := x/c$ 와 같이 치환하면 $$ \begin{eqnarray*} \left| D_{c} f \right|_{2}  &=& \int_{-\infty}^{\infty} \left| D_{c} f \left( x \right) \right|^{2} dx  \\ &=& \int_{-\infty}^{\infty} \left| {{ 1 } \over { \sqrt{c} }} f \left( {{ x } \over { c }} \right)  \right|^{2} dx \\ &=& \int_{-\infty}^{\infty} {{ 1 } \over { c }} \left| f \left( t \right) \right|^{2} c dt \\ &=& \int_{-\infty}^{\infty} \left| f \left( t \right) \right|^{2} dt \\ &=& \left| f \right|_{2} \end{eqnarray*} $$ 이므로 $D_{c}$ 는 바운디드다.
