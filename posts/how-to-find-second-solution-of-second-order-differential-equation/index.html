2계 미분방정식의 두 번째 해를 구하는 방법

553 1계·2계 미분 방정식  작성일: 0001-01-01 
작성자: 전기현 사장  

주어진 미분방정식이 $y^{\prime \prime }+p(t)y^\prime + q(t)y=0$ $\cdots \ \ (1)$이고, 하나의 해를 알고 있다고 하자.그 해를 $y_1$이라고 하면 두 번째 해를 찾는 방법은 아래와 같다. 일반해를 $y(t)=\nu(t) y_1(t)$라고 가정하자.$y$의 1계, 2계 미분을 구해보면$y^\prime = \nu ^\prime y_1 + \nu y_1^\prime $$ y^{\prime \prime} = \nu ^{\prime \prime}y_1 + \nu ^\prime y_1^\prime + \nu^ \prime y_1^\prime + \nu y_1^{\prime \prime} = \nu ^{\prime \prime}y_1 + 2\nu ^\prime y_1^\prime + \nu y_1^{\prime \prime}$ 구한 $y^\prime$, $y^{\prime \prime}$을 식 $(1)$에 대입하면,$\nu^{\prime \prime}y_1 + 2\nu ^\prime y_1^\prime + \nu y_1 ^{\prime \prime} + p \left( \nu ^\prime y_1 + \nu y_1^\prime \right) + q\nu y_1=0$ $\nu$에 대해서 정리하면,$\nu \left( y_1 ^{\prime \prime} + py_1 ^\prime + qy_1 \right) + \nu ^\prime \left( 2y_1 ^\prime + py_1 \right) + \nu ^{ \prime \prime} y_1=0$ $\cdots \ \ (2)$이 때 $y_1$은 식 $(1)$의 해이므로 $y_1^{\prime \prime} + py_1^ \prime + qy=0$이다.따라서 식$(2)$의 첫 번째 항은 $0$이고 다시 정리하면$\nu ^\prime \left( 2y_1 ^\prime + py_1 \right) + \nu ^{ \prime \prime}y_1=0$ 이 때 미분방정식의 계수를 낮추기 위해서 $\nu^\prime \equiv w$라고 치환하자.그러면 식은 아래와 같이 1계 미분방정식이 되어 계수가 낮아진다.$w \left( 2y_1^\prime+ py_1 \right) + w^\prime y_1=0$ 변수분리 등을 이용해서 새로이 얻은 $w$에 대한 미분방정식을 풀면 두 번째 해와 일반해를 얻을 수 있다.예제를 통해 구체적으로 알아보자. 예제  $2t^2 y^{\prime \prime} + 3ty^\prime –y=0$, $t>0$, $y_1=t^{-1}$일 때 두 번째 해와 일반해를 구하라.

$y=\nu t^{-1}$이라고 하면$y^\prime = \nu ^\prime t^{-1} - \nu y^{-2} $$ y^{\prime \prime } = \nu ^{\prime \prime} t^{-1} - 2v^\prime t^{-2}+ 2\nu t^{-3}$ 주어진 미분방정식에 대입하면$2t^2 \left( \nu ^{\prime \prime} t^{-1} + 2\nu t^{-3} - 2\nu ^\prime t^{-2} \right) +3t \left( \nu ^\prime t^{-1} - \nu t^{-2} \right) -\nu t^{-1} = 0 $$ \nu$ 에 대해서 정리하면 $\nu$항은 $0$이므로 식은$2t\nu^{\prime \prime} -\nu ^\prime =0$ 이 때 $\nu ^\prime \equiv w$라고 치환하면$2tw^\prime –w=0 $$ \Rightarrow 2t w^\prime =w $$ \Rightarrow \dfrac{1}{w}dw=\dfrac{1}{2t}dt $$ \Rightarrow \ln w = \dfrac{1}{2} \ln t + C = \ln t ^{1/2} +C $$ \Rightarrow w = Ce^{\ln t^{1/2}} = Ct^{1/2} $$ \Rightarrow \nu ^\prime=w=Ct^{1/2} $$ \Rightarrow \nu = \frac{2}{3}Ct^{\frac{3}{2}} + k$ $y=\nu y_1=\nu t^{-1}$이므로$y=\left( \frac{2}{3}Ct^{\frac{3}{2}} + k \right) t^{-1} $$ \Rightarrow y=\frac{2}{3}Ct^{\frac{1}{2}} + kt^{-1}$
