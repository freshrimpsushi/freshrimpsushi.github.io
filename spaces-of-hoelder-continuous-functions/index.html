횔더 연속 함수 공간

1087 함수해석  작성일: 0001-01-01 
작성자: 전기현 사장  

목차

연속 함수 공간spaces\ of\ continuous\ functions 열린 집합 $\Omega \subset \mathbb{R}^n$가 있다고 하자. 음이 아닌 정수 $m$에 대해서 $C^m(\Omega)$를 아래와 같은 조건을 만족시키는 모든 함수 $\phi$들의 집합이라고 정의한다.$|\alpha| \le m$인 모든 멀티 인덱스 $\alpha$에 대해 $D^{\alpha}\phi$가 $\Omega$에서 연속이다.그리고 특별히 $C^0(\Omega)=C(\Omega)$, $C^{\infty}=\cap_{m=0}^{\infty}C^m(\Omega)$로 정의한다. 또한 $C^m_c(\Omega) \subset C^m(\Omega)$는 $C^m(\Omega)$의 원소들 중 $\Omega$에서 컴팩트 서포트를 가지는 모든 원소들의 집합이다.**유계 연속 함수 공간**spaces\ of\ bounded,\ continuous\ functions 위의 연속 함수 공간의 정의에 따르면 $\phi$가 $\Omega$ 위에서 유계일 필요는 없다. 따라서 $C^m_B(\Omega)$를 모든 $0 \le |\alpha| \le m$에 대해 $D^\alpha \phi$가 $\Omega$ 위에서 유계인 $\phi \in C^m(\Omega)$들의 집합으로 정의한다. 그러면 $C^m_B(\Omega)$는 아래와 같은 놈이 주어진 바나흐 공간이다. $$ \left| \phi\ ;\ C^m_B( \Omega )\right| = \max \limits_{0 \le |\alpha | m } \sup \limits_{x\ \in \Omega} | D^{\alpha}\phi(x) | $$ 유계 균등 연속 함수 공간****spaces\ of\ bounded,\ uniformly\ continuous\ functions $\phi \in C(\Omega)$가 $\Omega$ 위에서 유계이고 균등하게 연속이라고하자. 그러면 $\Omega$의 **클로저**closure $\bar{\Omega}$에서도 유계이고 연속이다. 따라서 $C^m (\bar{\Omega} ) $를 모든 $0 \le |\alpha| \le m$에 대해 $D^\alpha \phi$가 $\Omega$ 위에서 유계이고 균등하게 연속인 $\phi \in C^m(\Omega)$들의 집합으로 정의한다. $C^m( \bar \Omega )$는 $C^m_B(\Omega)$의 닫힌 부분공간이고 같은 놈이 주어지는 바나흐 공간이다. $$ \left| \phi\ ;\ C^m(\bar \Omega )\right| = \max \limits_{0 \le |\alpha | m } \sup \limits_{x\ \in \Omega} | D^{\alpha}\phi(x) |  $$ **횔더 조건**Hoelder\ condition **아래의 식을 만족시키는 상수 $K$가 존재하면 횔더 조건을 만족시킨다고 한다.** $$ \left| D^{\alpha} \phi(x) - D^{\alpha}\phi(y) \right| \le K |x-y|^\lambda,\quad \forall\ x,y \in \Omega $$ **** **** **횔더 연속 함수 공간**spaces\ of\ Hoelder\ continuous\ functions $ 0 \le \lambda \le 1$이라고 하자. 그러면 $C^{m,\lambda}(\bar \Omega) \subset C^m(\bar \Omega) $를 모든 $0 \le |\alpha| \le m$에 대해 $D^\alpha \phi$가 $\Omega$에서 $\lambda$에 대한 횔더 조건을 만족시키는 $\phi \in C^m(\Omega)$들의 집합으로 정의한다. 그러면 $C^{m,\lambda}( \bar \Omega )$는 다음과 같은 놈이 주어진 바나흐 공간이 된다. $$ \left| \phi\ ;\ C^{m,\lambda} (\bar \Omega )\right| =\left| \phi\ ;\ C^{m} (\bar \Omega )\right| +  \max \limits_{0 \le |\alpha | m } \sup \limits_{x,y\ \in \Omega \\ x\ne y } \dfrac{ | D^{\alpha}\phi(x) - D^{\alpha}\phi(y) |}{ |x-y| ^{\lambda}} $$ 또한 $0 < \nu < \lambda \le 1$에 대해서 아래의 관계가 성립한다. $$ C^{m,\lambda}( \bar \Omega ) \subset C^{m,\nu}( \bar \Omega ) \subset C^m( \bar \Omega ) $$
