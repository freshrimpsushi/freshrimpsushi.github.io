스튜던트의 정리Students Theorem 증명

203 수리통계학  작성일: 0001-01-01 
작성자: 류대식 사장  

$X_{i} \sim N(\mu,\sigma)$이라고 하자.**(a)** $\displaystyle \overline{X} \sim N\left( \mu , { {\sigma^2} \over {n} } \right) $**(b)** $\overline{X} \perp S^2$**(c)** $\displaystyle (n-1) { {S^2} \over {\sigma^2} } \sim \chi^2 (n-1) $**(d)** $\displaystyle  T = { {\overline{X} - \mu } \over {S / \sqrt{n}} } \sim t(n-1)$통계학을 하는 사람들은 당연한듯 쓰고 있지만 사실 여기에도 이름이 있다.네 개의 파트로 나뉘어져 있어 구체적으로 인용하기도 어려운 것도 한몫을 한 것 같다.**증명(a)** $\displaystyle \overline{X} = { { (X_1 + X_2 + \cdots + X_n )} \over {n}}$이므로,정규분포를 따르는 확률변수들의 합을 생각해보면 $\displaystyle \overline{X} \sim N(\mu, {{1} \over {n}} \sigma^2 )$

■

**증명(b)** 당장은 어렵기 때문에 생략한다.간단한 개요만 남기자면, $S^2$의 분포가 $\mu$에 무관함을 보인 후 바수의 정리를 사용한다.

■

순서통계량을 배워야 증명을 이해할 수 있다.바수의 정리가 블로그에 업데이트 되면 자세한 증명을 남기겠다.**증명(c)** $\displaystyle V = \sum_{i=1}^{n} \left( { {X_{i} - \mu } \over {\sigma} } \right) ^2 $ 라고 하자.$\displaystyle { {X_{i} - \mu } \over {\sigma} } \sim N(0,1)$ 이므로 $V \sim \chi^2 (n)$일 것이다.$\displaystyle V = \sum_{i=1}^{n} \left( { {X_{i} - \mu } \over {\sigma} } \right) ^2 $$ \displaystyle = \sum_{i=1}^{n} \left( { { ( X_{i} -\overline{X} ) + ( \overline{X} - \mu ) } \over {\sigma} } \right) ^2 $$ \displaystyle = \sum_{i=1}^{n} \left( { {  X_{i} -\overline{X} } \over {\sigma} } \right) ^2 + \left( { { \overline{X} - \mu } \over {\sigma / \sqrt{n} } } \right) ^2 $여기서 $\displaystyle \sum_{i=1}^{n} \left( { {  X_{i} -\overline{X} } \over {\sigma} } \right) ^2 = { {n-1} \over {\sigma^2} } \sum_{i=1}^{n} { { ( X_{i} -\overline{X} ) ^ 2 } \over {n-1} } = (n-1) { {S^2} \over {\sigma^2} } $정리하면 $\displaystyle V = (n-1) { {S^2} \over {\sigma^2} }  + \left( { { \overline{X} - \mu } \over {\sigma / \sqrt{n} } } \right) ^2 $$ V \sim \chi^2 (n)$이고 (a)에 의해 $\displaystyle  \left( { { \overline{X} - \mu } \over {\sigma / \sqrt{n} } } \right) \sim N(0,1)$ 이고, 표준정규분포의 제곱은 카이제곱분포를 따르므로 $\displaystyle  \left( { { \overline{X} - \mu } \over {\sigma / \sqrt{n} } } \right)^2 \sim \chi^2 (1)$(b)에서 $\overline{X}$와 $S^2$이 서로 독립임을 보였으므로, 양변에 적률생성함수를 취하면$\displaystyle (1-2t)^{-n/2} = E \left\{  \exp \left( (n-1) { {S^2} \over {\sigma^2} } t \right) \right\} (1-2t)^{-1/2}$따라서 $\displaystyle (n-1) { {S^2} \over {\sigma^2} }$의 적률생성함수는 $ (1-2t)^{-(n-1)/2}$

■

**증명(d)** **정규분포와 카이제곱분포에서 스튜던트 t분포 유도** $W \sim N(0,1)$이고 $V \sim \chi^2 (r)$이면 $\displaystyle T = { {W} \over {\sqrt{V/r} } } \sim t(r) $$ \displaystyle T = { {\overline{X} - \mu } \over {S / \sqrt{n}} } = { {( \overline{X} - \mu ) / (\sigma / \sqrt{n}) } \over { \sqrt{ (n-1) S^2 / ( \sigma^2 ( n-1 ) ) } } } $(a)에서 $\displaystyle \overline{X} \sim N\left( \mu , { {\sigma^2} \over {n} } \right) $이고 (c)에서 $\displaystyle (n-1) { {S^2} \over {\sigma^2} } \sim \chi^2 (n-1) $임을 보였으므로 $T \sim t(n-1)$

■
