작성자: 류대식 사장  

목차

$E$ 가 $\mathbb{R}^{n}$ 에서 오픈이고 $ \mathbb{f} \in C^{1} (E)$ 와 $\phi_{0} \in E$ 에 대해 초기값 문제 $\begin{cases} \dot{ \phi } = \mathbb{f} ( \phi )  \\ \phi (0) = \phi_{0} \end{cases}$ 가 있다고 하자. 그러면 어떤 $[ - h , h ]$ 에서 주어진 초기값 문제의 솔루션 $\phi( t )$ 은 유일하게 존재한다.* 유클리드 공간 $\mathbb{R}^{n}$ 의 볼을 $B \left( \mathbb{x}_{0} ; d \right) := \left\{ \mathbb{x} \in \mathbb{R}^{n} \mid | \mathbb{x}_{0} - \mathbb{x} | < d \right\}  $, $B \left[ \mathbb{x}_{0} ; d \right] = \left\{ \mathbb{x} \in \mathbb{R}^{n} \mid | \mathbb{x}_{0} - \mathbb{x} | \le d \right\}  $ 과 같이 표현한다.당연히 좀 더 강한 조건으로써 $\mathbb{f} \in C^{1} (E)$ 대신 립시츠 조건이 있어도 무방하다.**증명** 존재성을 먼저 보인 후 유일성을 보이겠다.**** **Part 1.** 로컬리 립시츠$\mathbb{f} \in C^{1} (E)$ 면 $\mathbb{f}$ 는 $E$ 에서 로컬리 립시츠다.$\mathbb{f}$ 가 $E$ 에서 로컬리 립시츠이므로, 모든 $\mathbb{x} , \mathbb{y} \in B \left( \phi_{0} ; \varepsilon \right) \subset E$ 에 대해 $ | \mathbb{f} ( \mathbb{x} ) - \mathbb{f} ( \mathbb{y} ) | \le K | \mathbb{x} - \mathbb{y} | $ 를 만족하는 $\varepsilon , K > 0 $ 가 존재한다.컴팩트와 연속$f : X \to Y$ 에 대해 $X$ 가 컴팩트, $f$ 가 연속이면 $f(X)$ 는 컴팩트다.$\mathbb{f}$ 는 연속이므로 컴팩트셋 $B : = B \left[ \phi_{0} ; {{ \varepsilon } \over {2}} \right] $ 에서 바운디드고, $\displaystyle M : = \sup_{ \mathbb{x} \in B } | \mathbb{f} ( \mathbb{x} ) |$ 를 잡을 수 있다.**Part 2.** 피카드 메소드$E$ 가 $\mathbb{R}^{n}$ 에서 오픈이고 $ \mathbb{f} \in C^{1} (E)$ 에 대해 초기값 문제 $\begin{cases} \dot{ \phi } = \mathbb{f} ( \phi )  \\ \phi (0) = \phi_{0} \end{cases}$ 가 있다고 하자. 함수의 시퀀스 $\left\{ \mathbb{u}_{k} (t) \right\} _{ k =0}^{ \infty }$ 을 $\begin{cases} \mathbb{u}_{0} (t) = \phi_{0}  \\ \displaystyle \mathbb{u}_{k+1} (t) = \phi_{0} + \int_{0}^{t} \mathbb{f} \left( \mathbb{u}_{k} (s) \right) ds \end{cases}$ 과 같이 정의하면 연속함수 $\displaystyle \mathbb{u} (t) := \lim_{k \to \infty} \mathbb{u}_{k} (t) $ 는 주어진 초기값 문제의 솔루션이다.연속함수 $\mathbb{u}_{k} (t) $ 가 $\displaystyle \sup_{ t \in [ -h , h ] } | \mathbb{u}_{k} (t) - \phi_{0} | \le {{\varepsilon } \over {2 }}$ 를 만족시면서 $\begin{cases} \mathbb{u}_{0} (t) = \phi_{0}  \\ \displaystyle \mathbb{u}_{k+1} (t) = \phi_{0} + \int_{0}^{t} \mathbb{f} \left( \mathbb{u}_{k} (s) \right) ds \end{cases}$ 와 같이 정의된다고 가정해보자.$\mathbb{u}_{k} $ 와 $\mathbb{f}$ 는 $[-h,h]$ 에서 연속이므로 $( \mathbb{f} \circ \mathbb{u}_{k} ) $ 역시 연속이다.미적분학의 기본정리함수 $f$ 가 폐구간 $[a,b]$ 에서 연속이면 함수 $\displaystyle F(x) = \int_{a}^{x} f(t) dt$ 는 $[a,b]$ 에서 연속, $(a,b)$ 에서 미분가능하며 $\displaystyle {{dF(x)} \over {dx}} = f(x)$그러면 미적분학의 기본정리에 의해 $\displaystyle \mathbb{u}_{k+1} (t) = \phi_{0} + \int_{0}^{t} \mathbb{f} \left( \mathbb{u}_{k} (s) \right) ds$ 역시 $[-h, h]$ 에서 연속이다.식을 살짝 정리해서 부등식을 세우면 모든 $t \in [-h, h]$ 에 대해$\displaystyle | \mathbb{u}_{k+1} (t) - \phi_{0} | \le \int_{0}^{t} \left| \mathbb{f} \left( \mathbb{u}_{k} (s) \right) \right| ds $$ \displaystyle \le \int_{0}^{h} \left| \mathbb{f} \left( \mathbb{u}_{k} (s) \right) \right| ds $$ \displaystyle \le \int_{0}^{h} M ds \le Mh$다시 말해, $\displaystyle h \in \left( 0 , {{ \varepsilon } \over { 2M }} \right] $ 를 선택함으로써 $\mathbb{u}_{k} (t)$ 는 모든 $t \in [-h,h]$ 와 $k = 1,2,3 \cdots$ 에서 연속함수로 정의될 수 있다.**Part 3.**  코시 수열 $\left\{ \mathbb{u}_{k} \right\}_{k=0}^{\infty} $$ t \in [-h , h ]$ 에 대해 $ | \mathbb{ u }_{ j + 1 } - \mathbb{ u }_{ j } | $ 의 상한을 계산할 것이다.**Case 1.** $j = 1 $$ | \mathbb{ u }_{ 2 } ( t ) - \mathbb{ u }_{ 1 } ( t ) |  $$ \displaystyle \le \int_{0}^{t} \left| \mathbb{f} \left( \mathbb{u}_{1} (s) \right) - \mathbb{f} \left( \mathbb{u}_{0} (s) \right) \right| ds $$ \displaystyle \le K \int_{0}^{t} | \mathbb{ u }_{ 1 } ( s ) - \mathbb{ u }_{ 0 } ( s ) | ds $$ \displaystyle \le K h \sup_{ t  \in [ - h , h ] }  | \mathbb{ u }_{ 1 } ( t ) - \phi_{ 0 } | $$ \displaystyle \le {{ K h \varepsilon } \over { 2 }}$**Case 2.**  $j > 1 $$ | \mathbb{ u }_{ j+1 } ( t ) - \mathbb{ u }_{ j } ( t ) |  $$ \displaystyle \le \int_{0}^{t} \left| \mathbb{f} \left( \mathbb{u}_{ j } (s) \right) - \mathbb{f} \left( \mathbb{u}_{ j - 1 } (s) \right) \right| ds $$ \displaystyle \le K \int_{0}^{t} | \mathbb{ u }_{ j } ( s ) - \mathbb{ u }_{ j - 1 } ( s ) | ds $$ \displaystyle \le K h \sup_{ t  \in [ - h , h ] }  | \mathbb{ u }_{ j  } ( t ) - \mathbb{u}_{j-1} ( t ) |$재귀적으로 풀어내면 **Case 1.** 에 의해 $\displaystyle | \mathbb{ u }_{ j+1 } ( t ) - \mathbb{ u }_{ j } ( t ) | \le {{ (Kh)^{j} \varepsilon } \over {2}} $$ m > k > N$ 과 $\displaystyle h \in \left( 0 , {{1} \over { K }} \right)$ 이라 하고 $c := K h$ 라고 두면$ | \mathbb{ u }_{ m } ( t ) - \mathbb{ u }_{ k } ( t ) |  $$ \displaystyle \le \sum_{j = k}^{m-1} | \mathbb{u}_{j+1} (t) - \mathbb{u}_{j} (t) | $$ \displaystyle \le \sum_{j = N}^{ \infty } | \mathbb{u}_{j+1} (t) - \mathbb{u}_{j} (t) | $$ \displaystyle \le \sum_{j = N}^{ \infty } {{ ( K h )^{ j } \varepsilon } \over {2}} = {{ c^{N} } \over { 1 - c}} {{ \varepsilon } \over {2}} $$ |c| < 1$ 이므로 $N \to \infty$ 일 때 $\displaystyle {{ c^{N} } \over { 1 - c}} {{ \varepsilon } \over {2}}$ 은 $0$ 으로 수렴한다.다시말해, 모든 $\varepsilon > 0$ 에 대해 $m , k \ge N \implies $$ \displaystyle | \mathbb{u}_{m} - \mathbb{u}_{k} | = \sup_{ t \in [-h , h ] } | \mathbb{u}_{m} ( t ) - \mathbb{u}_{k} ( t ) | < \varepsilon $ 을 만족하는 $N$ 이 존재한다.이는 $\left\{ \mathbb{u}_{k} \right\}_{k=0}^{\infty} $ 이 $C [ - h , h ] $ 의 코시 수열임을 의미한다.(물론 여기서 선택되는 $h$ 는 $\displaystyle  0 < h < \min \left( {{b} \over {M}} , {{1} \over {K}} \right)$ 을 만족해야한다.)**Part 4.**  바나흐 공간$C [ -h , h ] $ 는 바나흐 공간이므로 $\displaystyle \mathbb{u} (t) := \lim_{k \to \infty} \mathbb{u}_{k} (t) $ 는 연속함수다.$\mathbb{u} $ 가 연속이므로 $( \mathbb{f} \circ \mathbb{u} ) $ 역시 연속이고, 미적분학의 기본정리에 의해 $\displaystyle \dot{\mathbb{u} } (t) = \left( \phi_{0} + \int_{0}^{t}  \mathbb{f} \left( u(s) \right) ds \right)' = \mathbb{f} \left( \mathbb{u} (t) \right)$또한 $t = 0$ 이면 $\displaystyle \mathbb{u} (0) = \phi_{0} + \int_{0}^{0}  \mathbb{f} \left( u(s) \right) ds = \phi_{0} + 0 = \phi_{0}$따라서 $\mathbb{u}$ 는 모든 $t \in [ - h , h ]$ 에 대해 주어진 초기값 문제의 솔루션으로써 존재한다.**Part 5.** 유일성$\mathbb{u}$ 와 $\mathbb{v}$ 가 주어진 초기값 문제의 솔루션이라고 가정하자.$| \mathbb{u} (t) - \mathbb{v} (t) | $ 가 가장 큰 값을 가지도록 하는 $t$ 를 $t_{0} \in [-h , h]$ 이라고 하면$\displaystyle | \mathbb{u} - \mathbb{v} | = \sup_{t \in [-h,h ] } | \mathbb{u} (t) - \mathbb{v} (t) | $$ \displaystyle = \left| \int_{0}^{t_{0} } \left[ \mathbb{f} \left( \mathbb{u} (s) \right) - \mathbb{f} \left( \mathbb{v} (s) \right) \right] ds \right| $$ \displaystyle \le  \int_{0}^{t_{0} } \left| \mathbb{f} \left( \mathbb{u} (s) \right) - \mathbb{f} \left( \mathbb{v} (s) \right) \right| ds $$ \displaystyle \le  K \int_{0}^{t_{0} } \left|\mathbb{u} (s) - \mathbb{v} (s) \right| ds $$ \displaystyle \le  K h \sup_{s \in [-h, h ] } \left| \mathbb{u} (s) - \mathbb{v} (s) \right|  $$ \le K h | \mathbb{u} - \mathbb{v} | $정리하면 $ | \mathbb{u} - \mathbb{v} | \le K h | \mathbb{u} - \mathbb{v} | $ 인데, $Kh < 1$ 이므로 $| \mathbb{u} - \mathbb{v} | = 0$ 이어야한다.따라서 $[-h, h ]$ 에서는 $| \mathbb{u} | = | \mathbb{v} | $ 이어야한다.
