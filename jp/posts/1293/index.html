定理

アイゼンシュタイン環の既約元をアイゼンシュタイン素数と呼ぶ。アイゼンシュタイン整数 $\pi \in \mathbb{Z}[ \omega ]$ が以下のいずれかの条件を満たすとき、アイゼンシュタイン素数である。

  • (i): $\pi = 1 + \omega 2$
  • (ii): 素数 $p \in \mathbb{Z}$ に対し、$p \equiv 2 \pmod{3}$ の $\pi = p$
  • (iii): 素数 $p \in \mathbb{Z}$ に対し、$p \equiv 1 \pmod{3}$ の場合、$p = u^2 - uv+ v^2$ を満たす $\pi = u + \omega v$
  • (iv): (i)～(iii) に該当する $\pi$ に$\mathbb{Z} [\omega ]$ の単位 $\pm 1 , \pm \omega , \pm \omega^2$を乗じて得られる $ \pm \omega^{k} \pi$
  • (v): (i)～(iii) に該当する $\pi$ に共役を取って得られる $\overline{\pi}$

説明

アイゼンシュタイン整数について話すとき、$\pi$ は通常、混乱を避けるために円周率ではなく、アイゼンシュタイン素数を指す。これは、通常の整数の素数を自然素数natural Prime $p$ と呼ぶことが多いためである。このような素数の拡張は、アイゼンシュタイン整数に関する研究を数論らしくしてくれる。この点はガウス素数と似ている。

(i)

$(1 + \omega 2)$ は、$3 = - (1 + \omega 2)^2$ の代わりとして、$3 \in \mathbb{Z} [ \omega ]$ の素因数分解に必要な数である。素数$2 \in \mathbb{Z}$ はタイプ(ii)の素数であり、最小の素数とは言えないが、それは$\mathbb{Z}$ においても同様であった。

(ii)

例として$5$ は、アイゼンシュタイン整数を使っても素因数分解できない。実際にできないかを確認することは無意味なので、そのまま証明を見ることを推奨する。

(iii), (iv), (v)

例えば、$7$ は$7 = (3 + \omega)(2 - \omega )$ であるため、素因数分解が可能である。ここで、$\mathbb{Z}[ \omega ]$ はUFDであるため、唯一の素因数分解を持つ。一方、$(3 + \omega)$ は$2 - \omega = 3 - (1 + \omega) = 3 + \overline{\omega}$ と共役を成すことでアイゼンシュタイン素数になることが確認できる。

証明

戦略: $(Z[ \omega ] , N)$ は、$N(x+\omega y) = x^2 - xy + y^2$ として定義されたアイゼンシュタイン環のノルムである。（ただし、$x, y \in \mathbb{Z}$）アイゼンシュタイン素数定理の証明自体は、その代数的性質を持ち込んで初等整数論の様々な結果と組み合わせたものに過ぎないが、その代数的性質と様々な結果を理解することが難しい。

パート0. $| N( \pi ) | = p$ が素数なら、$\pi$ はアイゼンシュタイン素数である。

乗法的ノルムの性質: $ p \in \mathbb{Z}$ が素数だとする。

  • [1]: 乗法的ノルム$N$が$D$で定義されていると、$N(1) = 1$ であり、すべての単位$u \in D$に対して$| N ( u ) | = 1$
  • [2]: $| N ( \alpha )| =1$ を満たすすべての$\alpha \in D$ が$D$で単位ならば、$| N ( \pi ) | = p$ を満たす$\pi \in D$ は$D$で既約元である。

アイゼンシュタイン環の性質

  • [3]: $\mathbb{Z}[\omega]$ の単位は$\pm 1, \pm \omega , \pm \omega^2$ だけである。

[3]により、$\mathbb{Z}[ \omega ]$ の単位は$\pm 1, \pm \omega , \pm \omega^2$ だけであり、[1]により$N(\pm1) = N(\pm \omega) = N( \pm \omega^2 ) = 1$ が成り立つ。$| N ( \alpha )| =1$ を満たすすべての$\alpha$ が$\mathbb{Z}[ \omega ]$で単位であったため、[2]により、$| N( \pi ) | = p$ を満たす$\pi$ は$\mathbb{Z}[ \omega ]$の既約元となる。つまり、$| N( \pi ) | = p$ が素数なら、$\pi$ はアイゼンシュタイン素数である。

パート(i). $\pi = 1 + \omega 2$

$\pi = 1 + \omega 2$ ならば、$N(\pi) = 1^2 - 1 \cdot 2 + 2^2 = 3$ は素数であるため、パート0により、$\pi = 1 + \omega 2$ はアイゼンシュタイン素数である。

パート(ii). $\pi \equiv 2 \pmod{3}$

$\pi = p$ が$p \equiv 2 \pmod{3}$ を満たす$\mathbb{Z}$ の素数であるが、$\mathbb{Z}[ \omega ]$ のガウス素数ではないとして、$\pi = ( a + \omega b )( c + \omega d )$ のような素因数分解が存在すると仮定してみよう。$N$ の乗法的性質により、 $$ \begin{align*} p^2 =& \pi^2 - \pi \cdot 0 + ^2 \\ =& N ( \pi + \omega 0) \\ =& N ( a + \omega b ) N ( c + \omega d ) \\ =& (a^2 - ab+ b^2) (c^2 - cd + d^2) \end{align*} $$ となり、$p^2 = (a^2 - ab + b^2) (c^2 -cd + d^2)$ が成り立つが、$p \in \mathbb{Z}$ は素数であるため、$\begin{cases} a^2 - ab + b^2 = p \\ c^2 - cd + d^2 = p \end{cases}$ を満たす解が存在しなければならない。

素数が3で割った余りが1になる必要十分条件: $p \ne 3$ が素数だとする。$p \equiv 1 \pmod{3}$ $\iff$、何かの$a,b \in \mathbb{Z}$ に対して$p = a^2 - ab + b^2$

しかし、$p \equiv 2 \pmod{3}$ であるため、素数が3で割った余りが1になる必要十分条件により、$\begin{cases} a^2 - ab + b^2 = p \\ c^2 - cd + d^2 = p \end{cases}$ を満たす解は存在せず、これは矛盾であるため、$\pi \equiv 2 \pmod{3}$ はガウス素数である。

パート(iii). $\pi = u + \omega v$

素数 $p \in \mathbb{Z}$ に対し、$p \equiv 1 \pmod{3}$ であるため、素数が3で割った余りが1になる必要十分条件に従い、 $$ N (\pi) = N ( u+ \omega v) = u^2 -uv + v^2 = p $$ これを満たす$\pi = u + \omega v$ はパート0によりガウス素数である。

パート(iv). $ \pm \omega^{k} \pi$

パート0では、$| N( \pi ) | = p$ が素数ならば、$\pi$ はアイゼンシュタイン素数であるとしたので、$k \in \mathbb{Z}$ に対し、 $$ \begin{align*} N( \pm \omega^{k} \pi ) =& N( \pm \omega^{k} ) N (\pi ) \\ =& 1 \cdot N (\pi ) \\ =& p \end{align*} $$ これを満たす$\pm \omega^{k} \pi $ もまたアイゼンシュタイン素数である。

パート(v). $\overline{\pi}$

アイゼンシュタイン整数に共役を取ると、 $$ \begin{align*} \overline{x + \omega y} =& x + \overline{\omega} y \\ =& x - (1 + \omega) y \\ =& (x-y) - \omega y \end{align*} $$ そして、パート0で$| N( \pi ) | = x^2 - xy + y^2 = p$ が素数ならば、$\pi = x + \omega y$ はアイゼンシュタイン素数であることが述べられているので、 $$ \begin{align*} N( \overline{\pi} ) =& N \left( (x-y) - \omega y \right) \\ =& (x-y)^2 - (x-y) \cdot (-y)+ (-y)^2 \\ =& x^2 - 2 xy + y^2 + xy - y^2 + y^2 \\ =& x^2 - xy + y^2 \end{align*} $$ これを満たす$\overline{\pi}$ もまたガウス素数である。
